# 🍽️ Delicious Bites -- Premium Restaurant Website

A modern, fully responsive restaurant website built with **HTML5**,
**CSS3**, and **Vanilla JavaScript**. Designed to showcase a luxury
dining experience with elegant UI, smooth animations, and interactive
features.

## 🌐 Live Demo

**Website:** https://luxury-resturant.vercel.app

## 📂 Repository

**GitHub:** https://github.com/caziiz/Luxury_Resturant

------------------------------------------------------------------------

## 💡 Project Goal

This project was created to demonstrate modern front-end web development
using only HTML, CSS, and JavaScript without relying on frameworks. It
focuses on responsive design, performance, accessibility, and creating a
premium user experience suitable for a luxury restaurant.

## 🛠️ Technologies Used

-   HTML5
-   CSS3
-   Vanilla JavaScript (ES6)
-   CSS Grid
-   Flexbox
-   Google Fonts

## 🌟 Features

-   Fully responsive design
-   Sticky navigation
-   Mobile hamburger menu
-   Interactive menu filtering
-   Hero section with animations
-   Animated statistics
-   Lightbox gallery
-   Testimonials slider
-   FAQ accordion
-   Reservation form with validation
-   Newsletter subscription
-   Contact form
-   Smooth scrolling
-   Lazy-loaded images
-   Accessibility support
-   Cross-browser compatibility

## 🚀 Quick Start

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Customize the content and images as needed.
4.  Deploy to any static hosting provider.

## 🔧 Customization

-   Replace images in the `images` folder.
-   Update restaurant information.
-   Modify colors through CSS variables.
-   Edit menu items in `script.js`.
